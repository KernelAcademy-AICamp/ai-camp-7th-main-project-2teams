# 사업성 검토 — 북마크 AI 관리 서비스

**버전**: v1.0
**작성일**: 2026-07-09
**범위**: `scripts/prd.md` §4(OKR)·§6(가치제안)·§7.5(가정)에 흩어진 비용/수익/리스크를 재무 타당성 문서로 통합. Lean Canvas/GTM 부록 실체 파일은 리포에 없어 본 문서에서 신규 구성.

---

## 1. 목적

서비스의 경제적 효용(비용 구조, 수익 모델, 손익 가능성)을 판단하기 위한 단일 문서. PRD가 서술로 언급한 수치(유저당 비용 $0.02, MAU 500, 프리미엄 ₩4,900 가정 등)의 산출 근거를 명시하고, 검증되지 않은 가정은 가정으로 표시한다.

---

## 2. 비용구조

### 2.1 AI 비용 (변동비)

| 항목                   | 특성                          | 유저당 월 사용량(가정)          |
| ---------------------- | ----------------------------- | ------------------------------- |
| GPT-4o-mini 태깅       | 입력 저렴, 저장 건당 1회 호출 | 20건 저장/월                    |
| text-embedding-3-small | 매우 저렴, 저장 건당 1회 호출 | 20건 저장/월                    |
| **합계**               |                               | **~$0.02/유저/월 (PRD 인용치)** |

**주의**: 위 수치의 정확한 토큰 단가·평균 콘텐츠 길이 계산식은 PRD/리포 어디에도 상세 로그가 없다. $0.02는 PRD가 제시한 결론치를 그대로 인용한 것이며, 실사용 트래픽(A5/A29 로그) 기반 실측 재계산이 필요하다.

### 2.2 인프라 비용 (고정비)

- Vercel: 서버리스 함수 실행 — 무료 티어 한도 내 예상(500 MAU 기준 트래픽 낮음)
- Supabase: DB + pgvector + Auth — 무료 티어 용량 제한 도달 시점 미산정(후속 필요)

### 2.3 손익 구조 요약

유저당 비용이 극히 낮음($0.02/월) → 무료 서비스로도 초기 지속 가능. 수익화는 프리미엄 기능(A64) 의존.

## 6. 손익분기 간이 시뮬레이션

```
가정: MAU 500명, 유저당 비용 $0.02/월 → 월 총비용 $10
프리미엄 전환율 가정 시:
  전환율 5% (25명) × ₩4,900 = 월 매출 ₩122,500 (~$90) → 비용 상회
  전환율 1% (5명)  × ₩4,900 = 월 매출 ₩24,500  (~$18) → 비용 상회
```

유저당 비용이 극히 낮아 낮은 전환율에서도 손익분기 자체는 쉽게 넘긴다. **단, 이 계산은 A64(유일한 프리미엄 기능)가 구현·과금 인프라까지 완성된 것을 전제** — 현재 미구현 상태이므로 실질 매출은 $0이다.
