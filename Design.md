# 모와바 · 북마크 AI — Design System (Structured Teal)

브라우저 북마크를 저장하고 AI로 자동 분류·태깅하는 웹 서비스 **모와바(Mowaba)** 의 디자인 시스템 문서입니다.
본 문서는 채택된 방향인 **Structured Teal** 를 기준으로 작성되었습니다.

> 컨셉: 그라디언트 · 컬러코딩 · 밀도감. 딥틸→라이트틸 그라디언트를 브랜드 시그니처로 사용하고,
> 카테고리별 컬러코딩과 카드 상단 액센트 바로 정보 밀도가 높은 프로덕트 느낌을 만든다.

## 5. Components

### Top Bar
- 배경: 시그니처 그라디언트(135deg)
- 좌: 흰 라운드 칩(40px) 안에 심볼 마크 — **텍스트 없음**
- 우: `북마크 추가`(흰 버튼/틸 텍스트) · `파일 업로드`(반투명 고스트)

### Sidebar
- 배경 `#F8FAFB`, 우측 보더
- 탭 세그먼트(홈/즐겨찾기/내 폴더): 활성 = 딥틸 솔리드
- 카테고리 활성 항목 = 딥틸 솔리드 + 그림자, 나머지 = 컬러 도트 + 텍스트
- 하단 프로필: 그라디언트 아바타 + 모노 이메일 + 기어

### Bookmark Card
- 그라디언트 파비콘 스퀘어(모노 이니셜) + 카테고리 아웃라인 필 칩(30px)
- 제목 → 모노 도메인 → 태그(그레이 5px 칩) → 상단 보더 위 모노 날짜

### Buttons
- Primary: 그라디언트 배경 + 흰 텍스트 + 소프트 틸 그림자
- Secondary: 흰 배경 + 보더 + 슬레이트 텍스트
- Ghost(온 그라디언트): 반투명 흰 + 흰 보더

### Filters / Chips
- 검색바: 풀 라운드(30px) pill, `#F1F5F9` 배경, 틸 검색 아이콘
- 활성 태그 칩: 딥틸 솔리드 / 보조: 틸 틴트 아웃라인
- 정렬 `최신순 default` = 그라디언트, `유사도순` = 소프트블루 틴트

### Modal / Slide-over
- 18px radius
- 인풋 포커스: 2px 딥틸 보더 + `0 0 4px rgba(15,118,110,.1)` 링
- 미리보기 영역: 틸 틴트 카드(`#F0FDFA` / `#99F6E4`)

### Progress / States
- 프로그레스 바: 그라디언트 채움 위 `#E2E8F0` 트랙
- 진행 상태 카드: 좌측 3px 라이트틸 보더 + 회전 스피너
- 완료: 그라디언트 원형 체크 + 카테고리 컬러코딩 요약 리스트

## 6. Screens
1. 로그인 — 풀 로고 + 소셜 로그인(그라디언트 Google, 흰 카카오)
2. 메인 대시보드 — 사이드바 + 검색/정렬/뷰 토글 + 3열 카드 그리드
3. 북마크 추가 모달 — URL 입력 + 검증 미리보기 슬라이드오버
4. 파일 업로드 — 드래그앤드롭 임포트
5. 임포트 진행 — 업로드/검증/태깅 단계별 프로그레스
6. 임포트 완료 — 성공 메시지 + 카테고리 요약
7. 프로필 팝업 — 프로필 정보 / 설정 / 로그아웃

---

## 7. Principles
- **그라디언트는 브랜드 시그니처.** 상단바·프라이머리·강조 요소에만. 남발 금지.
- **컬러코딩으로 스캔 속도 향상.** 카테고리는 항상 지정된 색으로.
- **모노는 데이터에.** 도메인·날짜·버전·이메일 등 기계적 값에만 사용.
- **여백보다 밀도.** 타이트한 radius와 구분선/액센트 바로 정보 위계를 만든다.
- **성공/진행은 틸 계열로** 일관되게 — 별도 초록 대신 브랜드 그라디언트 사용.
